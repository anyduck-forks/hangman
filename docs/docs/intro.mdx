# Документацiя Hangman

Цей сайт мiстить документацiю до навчального проєкту "Шибениця" (Hangman) на React + Vite.

## Швидкий старт

```bash
npm install
npm run dev
```

## Документацiя

- Налаштування гри та iсторiя зберiгаються у localStorage.
- Згода на збереження даних запитується через cookie popup.
- Документи GDPR та Privacy Policy описують обробку даних i права користувача.

## Роздiли

- Користування застосунком
- Хуки
- Ентропiя та порiвняння лiтер
- Зберiгання даних i localStorage
