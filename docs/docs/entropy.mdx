# Ентропiя та порiвняння лiтер

У режимi аналiтики (`Info Theory Mode`) гра розраховує iнформацiйну кориснiсть кожної лiтери.
Мета — оцiнити, яка лiтера дасть найбiльше iнформацiї щодо прихованого слова.

## Кандидати

Для поточного стану (`wordProgress`, `guessedLetters`) формується регулярний вираз, який
вiдфiльтровує кандидати з `words_by_length.json`.

## Розподiл наслiдкiв

Для кожної лiтери `L` усi кандидати групуються за "маскою" збiгiв:

- Маска — це бiнарний рядок довжини слова.
- `1` означає, що лiтера `L` стоїть у цiй позицiї, `0` — нi.

Кожна маска має свою сумарну частоту (частоти слiв у словнику).

## Ентропiя

Для лiтери `L` обчислюється ентропiя Шеннона за формулою:

$$
H(L) = - \sum_i p_i \log_2 p_i
$$

де $p_i$ — iмовiрнiсть отримати маску $i$ при спробi лiтери $L$.

Бiльше $H(L)$ означає бiльше очiкуваної iнформацiї.

## Порiвняння

Гра показує ентропiї для всiх лiтер. Найбiльш "вигiдна" лiтера — та, що має найбiльшу ентропiю.
У застосунку це використовується для пiдсвiтки найбiльш iнформативних лiтер.

## Примiтка

Ентропiя обчислюється з урахуванням частот слiв, тому поширенi слова мають бiльший вплив,
нiж рiдкiснi.
