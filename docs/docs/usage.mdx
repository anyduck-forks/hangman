# 袣芯褉懈褋褌褍胁邪薪薪褟 蟹邪褋褌芯褋褍薪泻芯屑

## 小褌邪褉褌 谐褉懈

1. 袙i写泻褉懈泄褌械 谐芯谢芯胁薪褍 褋褌芯褉i薪泻褍.
2. 袨斜械褉i褌褜 褉i胁械薪褜 褋泻谢邪写薪芯褋褌i 邪斜芯 "Custom" 写谢褟 胁谢邪褋薪懈褏 锌邪褉邪屑械褌褉i胁.
3. 袧邪褌懈褋薪i褌褜 "Start Game".

## 校锌褉邪胁谢i薪薪褟 谐褉芯褞

- 袧邪褌懈褋泻邪泄褌械 谢i褌械褉懈 薪邪 泻谢邪胁i邪褌褍褉i, 褖芯斜 褉芯斜懈褌懈 褋锌褉芯斜懈.
- 袣i谢褜泻i褋褌褜 写芯褋褌褍锌薪懈褏 锌芯屑懈谢芯泻 蟹邪谢械卸懈褌褜 胁i写 芯斜褉邪薪芯谐芯 褉i胁薪褟 褋泻谢邪写薪芯褋褌i.

## 袪械蟹褍谢褜褌邪褌懈

袩i褋谢褟 蟹邪胁械褉褕械薪薪褟 谐褉懈 胁i写泻褉懈胁邪褦褌褜褋褟 械泻褉邪薪 褋褌邪褌懈褋褌懈泻懈 蟹:
- i褋褌芯褉i褦褞 褋锌褉芯斜
- 谐褉邪褎i泻芯屑 褉芯蟹锌芯写i谢褍 锌褉芯屑邪褏i胁
- 褕胁懈写泻懈屑懈 写i褟屑懈 "Play Again" 邪斜芯 锌芯胁械褉薪械薪薪褟屑 褍 屑械薪褞
