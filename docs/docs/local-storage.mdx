# Зберiгання даних у localStorage

Застосунок зберiгає данi локально у браузерi користувача.

## Ключi

- `hangman_user_id` — iдентифiкатор користувача.
- `hangman_game_history` — iсторiя iгор (результат, слово, кiлькiсть помилок, час, userId).
- Налаштування гри зберiгаються локально для зручностi.

## Згода

Cookie popup пояснює використання localStorage. Якщо користувач вiдхиляє, застосунок
використовує анонiмний userId `00000000-0000-0000-0000-000000000000` для iсторiї.

## Видалення

Користувач може видалити данi вручну через очищення localStorage або очищення даних сайту.
