# useCookieConsent

**Файл:** src/hooks/useCookieConsent.js

**Призначення:**
- Зберiгає згоду на використання localStorage та анонiмний режим.

**Повертає:**
- `consent` — стан згоди.
- `accept()`, `decline()`, `reset()`.
- `isAccepted`, `isDeclined`, `isUnknown`.
- `ANONYMOUS_UUID` — використовується у статистицi при вiдхиленнi згоди.
